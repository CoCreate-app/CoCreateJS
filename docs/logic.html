Pass Attributes

Passing attributes to elements can be done from same page or from page to new page. The attribute's can be applied to any element which will add a click event to the element.

<a pass-document_id="test" pass_to="A unique id"></a>

Logic Attributes

  • pass_to string required

    Template identifier. This attribute should be define.

  • pass_id string optional

    Collection name to fetch data. It is required.

  • pass-refresh string required

    Collection name to fetch data. It is required.

  • pass-collection string optional

    Collection name to fetch data. It is required.

  • pass-document_id string optional

    Define the field name to fetch array

    If this value defined, it will fetch the special field's value

  • pass-name string optional

    Define the field name to fetch array

    If this value defined, it will fetch the special field's value

  • pass-value string optional

    Define the value to filter by . Require fetch-name

    colleciton: whole collections list fetch.

  • pass-fetch-collection string optional

    collection

    colleciton: whole collections list fetch.

  • pass-fetch-name string optional

    collection

    colleciton: whole collections list fetch.

  • pass-filter-name string optional

    Collections's field name for filter.

  • pass-filter-value string optional

    Collection's field value for filter. If filter-name defined, this attribute should defined too
